# Лендинг SaaS-продукта (SCSS)

Проект создан в рамках изучения препроцессора SCSS (Sass). В работе реализована модульная структура стилей: файлы разбиты по секциям, используются переменные, миксины, вложенность и брейкпоинты для адаптива.

## Структура проекта
- `index.html` — главная разметка страницы.
- `scss/` — исходные файлы стилей, разбитые на модули (секции, переменные, миксины).
- `css/` — итоговый скомпилированный файл `styles.css`, который подключен к HTML.

## Как запускалась компиляция SCSS в CSS
В проекте использовался один из следующих методов:

**Вариант с Live Sass Compiler**
Сборка осуществлялась через расширение **Live Sass Compiler** (автор Glenn Marks). Для запуска использовалась кнопка `Watch Sass` на нижней панели редактора, которая автоматически отслеживала изменения в файлах `.scss` и компилировала их в `css/styles.css`.
